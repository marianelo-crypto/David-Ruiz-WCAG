Principios de accesibilidad para el contenido web

  • Perceptible
  • Operable
  • Comprensible
  • Robusto

Perceptible

El principio de percepción asegura que todas las personas puedan entender el contenido en línea. Esto implica ofrecer alternativas claras, como texto para imágenes y subtítulos en videos, para que todas las personas puedan acceder y comprender la información sin problemas.

Operable

El principio de operabilidad se trata de hacer que los sitios web sean fáciles de usar para todas las personas. Esto significa que todas las funciones deben ser accesibles, sin importar la habilidad o el dispositivo que se utilice. El objetivo es que todas las personas puedan interactuar con el sitio de manera efectiva y sin dificultades.

Comprensible

El principio de comprensible implica que la información y el funcionamiento de los componentes visuales deben ser comprensibles para todas las personas, incluidas aquellas con diferentes niveles de experiencia, educación y discapacidad.

Robusto

El principio de robusto se refiere a la compatibilidad de la web con diferentes tecnologías y dispositivos, garantizando que el contenido sea presentado y funcione de manera confiable en una amplia variedad de situaciones y plataformas, incluidos navegadores web, sistemas operativos y dispositivos de asistencia.
